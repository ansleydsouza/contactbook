# Contact Book
 A MERN Stack Contact Book Web Application

## Tech Stack Used

## Usage
```
git clone https://github.com/ansleydsouza/contactbook.git
cd contactbook/server
npm install
npm run app
cd ../contactbook/client
npm install
npm start
```
